# ENGINE DISK

Modelagem física do disco de acreção ao redor do buraco negro.
Baseado no modelo analítico de Novikov-Thorne (1973) para discos finos relativísticos.
Calcula perfis de temperatura, fluxo radiativo e efeitos de Redshift/Doppler gravitacional para visualização correta (efeito arco-íris relativístico).
Não realiza simulação de fluidos, apenas o modelo estacionário para renderização.
