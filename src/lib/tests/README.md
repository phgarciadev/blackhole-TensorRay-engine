# CORE TESTS

Suíte de testes unitários fundamentais para o núcleo matemático.
Verifica a corretude das operações vetoriais (Minkowski dot product) e tensoriais.
Valida se as métricas implementadas (Kerr/Schwarzschild) satisfazem propriedades físicas básicas, como simetria do tensor e limites assintóticos.
Essencial para garantir que a base matemática da simulação não contenha erros silenciosos.
