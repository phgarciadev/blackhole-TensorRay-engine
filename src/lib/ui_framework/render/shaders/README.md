# UX LIB SHADERS

Shaders GLSL para renderização da Interface de Usuário.
`ui.vert`: Vertex shader simples para transformar quads 2D (posição + UV).
`ui.frag`: Fragment shader que lida com texturas, cores sólidas e renderização de fontes (texture sampling).
Compilados para SPIR-V para uso pelo renderer Vulkan.
