# ENGINE GEODESIC

Integração numérica de trajetórias em espaço-tempo curvo.
Implementa o solver RK4 (Runge-Kutta de 4ª ordem) para a equação geodésica.
Suporta tanto geodésicas nulas (fótons/ray tracing) quanto tipo-tempo (partículas/órbitas).
Inclui utilitários para gerar raios a partir de uma câmera virtual e detectar colisões com o horizonte de eventos ou disco de acreção.
