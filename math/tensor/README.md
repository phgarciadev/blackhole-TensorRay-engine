# CORE TENSOR

Implementação de cálculo tensorial numérico para variedades Lorentzianas.
Define estruturas para o Tensor Métrico (`bhs_metric`) e Símbolos de Christoffel (`bhs_christoffel`).
Oferece funções para inversão de métrica, levantamento/abaixamento de índices e cálculo de aceleração geodésica (-Γ^α_μν u^μ u^ν).
Possui uma solver de Christoffel numérico via diferenças finitas para métricas arbitrarias.
