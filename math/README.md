# CORE MATH

Biblioteca de matemática fundamental focada em álgebra linear 4D para relatividade geral.
Implementa vetores `bhs_vec4` (espaço-tempo) e `bhs_vec3` (espacial auxiliar).
Destaque para produtos internos que respeitam a assinatura da métrica de Minkowski (-,+,+,+), fundamentais para distinguir intervalos tipo-tempo, tipo-espaço e nulos (fótons).
Não contém dependências externas, garantindo portabilidade.
