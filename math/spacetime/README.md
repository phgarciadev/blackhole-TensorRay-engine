# CORE SPACETIME

Implementações analíticas de métricas exatas da Relatividade Geral.
Atualmente suporta:
- Schwarzschild: Buracos negros estáticos (sem spin).
- Kerr: Buracos negros em rotação (solução completa de Boyer-Lindquist).
Fornece funções para calcular o tensor métrico em pontos específicos e determinar raios críticos (Horizonte de Eventos, Ergostera, ISCO, Esfera de Fótons).
Usado como "ground truth" para os cálculos numéricos.
