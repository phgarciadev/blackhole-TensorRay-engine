# Graphical User Interface

Responsabilidade: camada de interação e apresentação.
Agrega EPA (Engine Platform Abstraction), RHI (Rendering Hardware Interface, Vulkan/D3D/Metal/etc) e UI.
Cuida de janela, input, swapchain, eventos, e renderização final.
Ensina o sistema a renderizar, cuidar de telas e janelas.
Exporta biblioteca para construções de UIs, cenários, 3ds e outros. 
Não contém lógica de simulação nem física.
